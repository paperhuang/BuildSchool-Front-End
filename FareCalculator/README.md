> ### 預估交通費用計算

* 運用技術：HTML、CSS、BootStrap、JavaScript、Google Maps API
* 目的：讓使用者可以自行點選起點和終點，系統會計算出自行開車、搭計程車或是搭大眾運輸工具的價錢。
* 心得：這時候使用 Google Maps API 已經比較上手了，也學會按 F12 來看裡面的資料。
