> ### 我的蝦皮網站  

* 運用技術：HTML、CSS、BootStrap、套用 Font Awesome 的 icon
* 目的：練習網頁刻板，刻出一模一樣的蝦皮購物網站
* 心得：剛開始學習 Bootstrap，挑戰模仿別人的網站，雖然調整了很久，但是看著自己的網頁越來越像很有成就感。
